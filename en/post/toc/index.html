Vue2 cli安装

在创建项目之前需要先下载Node.js环境 可以利用npm安装淘宝镜像的cnpm，示例代码如下。

npm config set registry https://registry.npm.taobao.org

安装Vue脚手架

Vue CLI 3.x版本的包名称由vue-cli（旧版）改成了@vue/cli（新版），如果已经全局安装了旧版的vue-cli（1.x或2.x），需要通过如下命令进行卸载。

npm uninstall vue-cli -g

打开命令行工具，通过npm方式全局安装@vue/cli脚手架，具体命令如下。

npm install @vue/cli@3.10 -g

安装完成后，为了检测是否安装成功，使用如下命令来查看vue-cli的版本号。

vue -V     （或者vue --version）

打开命令行工具，使用vue create命令创建项目，它会自动创建一个新的文件夹，并将所需的文件、目录、配置和依赖都准备好。创建项目命令如下。

vue create vue_shop（项目名）

执行上述代码后，Vue CLI提示用户选取一个preset（预设），其中default是默认项，在这里选default即可，示例代码如下

Vue CLI v3.10.0
? Please pick a preset: (Use arrow keys)
> default (babel, eslint)
  Manually select features
